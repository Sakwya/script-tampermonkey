# PikPak分享占用空间查看  
>可以查看选中的文件占用的空间  

## 关于
PikPak每天只有三次免费，最多6G的传输，  
知道选中的文件到底占用了多少空间就显得十分重要。

## 功能  
- 选中文件时，可以显示占用的空间  
- 当选中多个文件时，可以显示累计占用的空间  
- 当选中内容包括文件夹时，会加上文件夹内所有文件的空间
